## ISSUE

### 1、关于mmu初始化顺序的理解

&emsp;曾经认为应该在拷贝代码之前初始化mmu,但是这样就使得链接地址失去了实际的意义。链接地址就是要把对应的代码拷贝到对应的存储位置。如果在拷贝代码之前初始化mmu，则代码将不能拷贝到对应的链接地址处。

### 2、mmu需要设置0x0段的地址映射

&emsp;因为在初始化mmu后，程序会返回到start.s中，而此时对应的pc地址还是启动时候的0x0开始的段，如果mmu没有这段映射，将无法运行跳转到main的这段代码。
